# Excel template (русские названия)

В этой папке лежит готовый шаблон как набор CSV-листов и собранный файл `template.xlsx`.

Листы в Excel должны называться:
- ДАННЫЕ_ЦЕНЫ
- ДОХОДНОСТИ
- СТАТИСТИКА
- ПОРТФЕЛЬ

Как использовать:
1) Откройте `excel_template/template.xlsx` и заполняйте лист `ДАННЫЕ_ЦЕНЫ` с 2-й строки.
2) Либо создайте новый Excel-файл и импортируйте CSV по листам.
3) Проверьте региональные настройки Excel: при необходимости замените запятые в формулах на точки с запятой.

Примечания:
- В формулах используются английские имена функций (AVERAGE, STDEV.S, CORREL, COVARIANCE.S, SQRT, MMULT).
- Названия активов (Актив_1..Актив_5) — это заглушки, их можно переименовать.
