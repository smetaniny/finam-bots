# Стратегия работы RTSM-3.26 (RMH6)

Основание: `Алгоритм разбора графика.md` и `Чек-лист разбора графика.md`.
Таймфреймы: D1 -> H4 -> H1. Торговля только на H1.
Период анализа для контекста: 2 месяца (2025-11-16 — 2026-01-15).

## 1) Контекст D1
- Определяем тренд по последовательности HH/HL или LL/LH.
- Важные экстремумы: отход минимум 1.5 * AvgRange (20 свечей).
- Строим 2–4 зоны D1 по важным экстремумам.

## 2) Зоны H4
- Строим зоны по тем же правилам (только важные экстремумы).
- Приоритет — пересечения зон D1 и H4.

## 3) Условия входа (H1)
Вход разрешен, если одновременно:
- Тренд D1 совпадает с направлением сделки (в боковике допускаются оба направления).
- Цена находится в зоне пересечения D1+H4.
- Есть свечной сигнал на H1.

Сигналы (по Валееву):
- Pin-bar (тень >= 2x тела, тень в сторону уровня).
- Key Reversal.
- Engulfing.

## 4) Вход и стоп
- Вход: по закрытию сигнальной H1 свечи.
- Стоп: за границей зоны (ниже поддержки / выше сопротивления).

## 5) Выход
- TP1: ближайшая противоположная зона H4.
- Остальное: трейлинг-стоп или по новым экстремумам.

## 6) Риск
- Депозит: 100 000 руб.
- Риск: 5% на сделку (5 000 руб).
- Риск фиксированный, без реинвестирования.
