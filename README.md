# Finam Bots MVP

Минимальный каркас для торгового бота под Finam API (REST).  
Фокус: разделение ядра, адаптеров и стратегий.

## Быстрый старт
1) Создай `.env` из `.env.example`
2) Установи зависимости:
   `pip install -r requirements.txt`
3) Запуск движка:
   `python main.py`

## Режимы
Режим задается в `configs/main.yaml`:
- `mode: "paper"` — бумажная торговля (без реальных ордеров)
- `mode: "live"` — реальные ордера (можно использовать демо-счет)

## Бэктест
Запуск исторического прогона:
`python backtesting/runner.py`

## Боевой запуск (Фаза 8)
1) Демо-режим 7 дней (mode: "live" + демо-аккаунт)
2) Проверка логов и стабильности
3) Реальный запуск с минимальными лимитами

## Визуализация (Streamlit)
Локально:
`streamlit run ui/app.py`

Панель показывает:
- свечной график
- позиции/сделки
- дневной и нереализованный PnL
- визуальные уровни стоп/тейк (если заданы в панели)
- последние логи

## Docker
Сборка и запуск:
`docker compose up --build`

Панель откроется на `http://localhost:8501`
